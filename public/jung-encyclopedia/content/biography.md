# Carl Jung（卡尔·荣格）

## 荣格是谁？

**卡尔·古斯塔夫·荣格（Carl Gustav Jung，1875—1961）** 是瑞士心理学家、精神病学家，也是分析心理学（Analytical Psychology）的创立者之一。

他曾与西格蒙德·弗洛伊德（Sigmund Freud）密切合作，但后来因理论分歧而发展出自己的心理学体系。

### 荣格提出的重要概念

荣格提出了许多影响深远的心理学概念，包括：

- **集体无意识（Collective Unconscious）**
- **原型（Archetypes）**
- **人格面具（Persona）**
- **阴影（Shadow）**
- **个体化（Individuation）**
- **心理类型（Psychological Types）**

### 与 MBTI 的关系

如今广为人知的 MBTI 与「荣格八维（八种认知功能）」都源于荣格于1921年出版的经典著作 **《Psychological Types（心理类型）》**。

## 荣格生平时间轴

| 年份 | 事件 |
|------|------|
| 1875 | 卡尔·古斯塔夫·荣格出生于瑞士凯斯维尔 |
| 1895 | 进入巴塞尔大学学习医学 |
| 1900 | 获得医学博士学位，开始精神病学研究 |
| 1907 | 与西格蒙德·弗洛伊德首次会面，开始密切合作 |
| 1912 | 出版《力比多的象征》，与弗洛伊德关系破裂 |
| 1913-1917 | "直面无意识"时期，进行自我实验和梦境分析 |
| 1921 | 出版《心理类型》，系统阐述心理类型理论 |
| 1933 | 担任苏黎世联邦理工学院心理学教授 |
| 1944 | 出版《心理学与炼金术》 |
| 1961 | 在瑞士苏黎世去世，享年85岁 |
