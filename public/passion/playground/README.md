# Playground — The Body's Expedition Log

Each sport is its own terrain. Drop your records here.

## Terrains
- `diving/` — Deep Memory Archive. Certificates, dive logs, underwater photos, fish records, depth records
- `skiing/` — Frost Ridge. Ski logs, mountain photos, gear config
- `climbing/` — The Ascent. Route records, grade progression, climbing photos
- `boxing/` — Underground Arena. Training logs, sparring footage, technique notes
- `skateboarding/` — Neon Drift. Trick progression, spot maps, session footage
- `certificates/` — Hall of Proof. All achievement certificates

**Supported**: JPG, PNG, PDF, MP4, TXT, MD
