# 📂 wsyverse 内容添加操作指南

## 🎯 核心原则

| 操作 | 允许 | 禁止 |
|------|------|------|
| 添加文件 | `public/` 下任意子文件夹 | ❌ 勿动 |
| 修改代码 | ❌ 勿动 | `src/`, `next.config.js`, `package.json` |
| 删除文件 | `public/` 里的内容文件 | ❌ `src/`, `node_modules/`, `.next/` |
| 重命名 | `public/` 里的文件 | ❌ 文件夹名称（含空格/中文） |

## 🔴 绝对不能动的文件夹/文件

```
wsyverse\
  ├── src\              ← ⛔ 代码核心，动了网站会崩
  ├── node_modules\    ← ⛔ 依赖库，删了项目跑不起来
  ├── .next\           ← ⛔ 构建缓存，自动生成
  ├── next.config.js    ← ⛔ 配置文件
  ├── package.json      ← ⛔ 项目配置
  ├── postcss.config.js ← ⛔ 样式配置
  ├── tailwind.config.ts← ⛔ 样式配置
  └── tsconfig.json     ← ⛔ TypeScript 配置
```

## 📋 添加内容的完整步骤

### 步骤 1：放入文件
把你的文件（PDF/图片/视频/文本）复制进对应的 `public\` 子文件夹。

### 步骤 2：部署到网上
在终端运行（需要你在电脑上操作）：
```bash
cd D:\wsyverse
vercel --prod
```
或者把改动的文件夹同步到 Git，Vercel 会自动重新部署。

### 步骤 3：检查网站
部署完成后，刷新你的网站 `https://wsyverse-xxx.vercel.app`，新内容会自动出现。

## 💡 小贴士

1. **文件名**：用英文字母/数字/下划线，避免中文和空格
2. **图片格式**：优先用 `.webp`（体积小），也可用 `.jpg` / `.png`
3. **PDF**：直接放进去，网站会自动生成封面图标
4. **笔记**：写 `.md` 文件（Markdown 格式），可以有格式
5. **不要**：在 `public\` 里创建新文件夹（除非你告诉我，我帮你配置路由）
